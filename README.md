# ToDoList on Spring Framework 

Actions:
- To build only - `./mvnw compile`
- To run tests - `./mvnw test`
- To build and run tests - `./mvnw install`
- To check codestyle - `./mvnw checkstyle:check`
- To run project - `./mvnw spring-boot:run`
